# Trading Strategy Research Lab

Repositorio de trabajo para extraer, comparar y destilar estrategias/métodos desde repos externos.

## Objetivo
No copiar bots completos.
Sí extraer:
- lógica de entrada
- lógica de salida
- control de riesgo
- timeframe
- supuestos de ejecución
- ideas reutilizables para:
  - XAU live lab
  - US500 cheap lab
  - executor layers sobre SE

## Método
1. Revisar repos listados en `repos_to_review.md`
2. Crear una strategy card por repo en `strategy_cards/`
3. Consolidar hallazgos en `comparisons/master_comparison.csv`
4. Extraer ideas accionables en `notes/` y `snippets/`
